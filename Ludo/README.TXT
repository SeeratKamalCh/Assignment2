Four player ludo game with single dice

Developed in python using tkinter

Run the main.py file to run the code

Enter your names and press start button to start

Roll dice button to roll dice

if all of your dummies are in home and dice<6 then it will automatically move to the next stage

if a 6 comes it will automatically roll again and give you accumulated ouput

On your turn, click on the dummy that you want to move after rolling dice
